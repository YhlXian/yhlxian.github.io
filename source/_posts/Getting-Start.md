## Setup

1. Setup Hexo
```bash
npm install hexo -g
```
1. Git Clone
```bash
git clone git@github.com:YhlXian/yhlxian.github.io.git
```
2. Add new post
```bash
hexo new 'post title'
```
3. Edit the post with markdown language
4. Post to Github
```bash
hexo g -d
```
